# Astropath *(working title)*

> An immersive **Warhammer 40k & Warhammer** audiobook + ebook library with full **Plex** integration, a cinematic cover-art experience, a living lore codex, and a **React Native (Expo + Tailwind)** iOS app.

Books and audiobooks in synergy — wrapped in cover-art, character bios, faction profiles and a navigable timeline that pulls you all the way into the setting. One curated, continually-updated home for the Warhammer reading + listening experience.

## What it is

- 🎧 **Audiobook player** — gapless, chaptered, background + lock-screen controls
- 📖 **EPUB library** — read the book or read along with the audio
- 🗄️ **Plex backend** — your server, your files, your library
- 🖼️ **Cinematic art** — large, captivating cover-art galleries on desktop
- 🧠 **Living codex** — characters, factions, events & planets with big artwork + sourced bios
- 🕰️ **Timelines** — every work pinned to where it falls in the grand timeline
- ✨ **Curated** — new releases with "where it fits," recommended orders, expert commentary

## Surfaces

- **iOS app** — React Native + Expo + NativeWind (Tailwind CSS)
- **Desktop / web** — art-forward, cinematic; the codex + cover-art experience at full size

Both share one backend and one data model.

## A note on content & IP

Designed as a **personal, self-hosted media companion** (the Plex model): you host the audiobooks/EPUBs you own; the app organises and enriches *your* library. Third-party artwork and lore are **fetched-on-demand, cached for personal use, and attributed/linked** — not bundled or redistributed. See [PLAN.md §9](PLAN.md#9-content-ip--legal-the-honest-bit).
